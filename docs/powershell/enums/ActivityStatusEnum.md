### ActivityStatusEnum
Represents activity statuses.

- Canceled - Canceled.
- Canceling - Canceling.
- Failure - Failure.
- Info - Information.
- PARTIAL_SUCCESS - Represents completion with warnings.
- Queued - Queued status.
- Running - Running.
- Success - Success.
- TaskFailure - Task failure status.
- TaskSuccess - Task success status.
- Warning - Warning.
