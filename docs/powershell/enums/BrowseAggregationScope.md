### BrowseAggregationScope
Scope for browse aggregation level.

- BROWSE_AGGREGATION_SCOPE_UNSPECIFIED - Unspecified aggregation scope.
- BROWSE_AGGREGATION_SCOPE_DIRECTORY - Aggregate results at the directory level.
- BROWSE_AGGREGATION_SCOPE_OBJECT - Aggregate results at the object level.
- BROWSE_AGGREGATION_SCOPE_FILE - Aggregate results at the file level.
