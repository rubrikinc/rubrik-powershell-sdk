### EksClusterAccessType
EKS cluster access type. The cluster can be either public or private, but
we will assume public if left unspecified.

- EKS_CLUSTER_ACCESS_TYPE_UNSPECIFIED - EKS cluster access type is not specified, defaults to public in application
logic.
- EKS_CLUSTER_ACCESS_TYPE_PRIVATE - EKS cluster will be private.
- EKS_CLUSTER_ACCESS_TYPE_PUBLIC - EKS cluster will be public.
