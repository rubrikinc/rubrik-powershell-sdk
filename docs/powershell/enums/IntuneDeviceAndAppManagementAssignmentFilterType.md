### IntuneDeviceAndAppManagementAssignmentFilterType
Specifies the type of an Intune device and app management assignment filter.

- INTUNE_DEVICE_AND_APP_MANAGEMENT_ASSIGNMENT_FILTER_TYPE_UNKNOWN - The type is unknown.
- INTUNE_DEVICE_AND_APP_MANAGEMENT_ASSIGNMENT_FILTER_TYPE_NONE - The type is none.
- INTUNE_DEVICE_AND_APP_MANAGEMENT_ASSIGNMENT_FILTER_TYPE_INCLUDE - The type is include.
- INTUNE_DEVICE_AND_APP_MANAGEMENT_ASSIGNMENT_FILTER_TYPE_EXCLUDE - The type is exclude.
