### HostConnectivityStatus
Connectivity status of a host.

- UNKNOWN_CONNECTIVITY_STATUS - Unknown connectivity status.
- CONNECTED - Host is connected.
- CONNECTING - Host is in the process of connecting.
- DISCONNECTED - Host is disconnected.
- REFRESHING - Host metadata is being refreshed.
- DELETING - Host is being deleted.
- BADLY_CONFIGURED - Host is badly configured.
- REMOTE - Host is on a remote cluster.
- CONNECTOR_NOT_DEPLOYED - Rubrik connector is not deployed on the host.
- DELETION_FAILED - Host deletion failed.
- REFRESH_FAILED - Host metadata refresh failed.
- PARTIALLY_CONNECTED - Host is partially connected.
- REPLICATED_TARGET - Host is a replicated target.
- DELETED - Host has been deleted.
- SECONDARY_CLUSTER - Host is on a secondary cluster.
- UNAUTHORIZED - Host is unauthorized.
