### IntuneComplianceActionType
Specifies the scheduled action type for device compliance.

- INTUNE_COMPLIANCE_ACTION_TYPE_UNKNOWN - The action type is unknown.
- INTUNE_COMPLIANCE_ACTION_TYPE_NO_ACTION - No action.
- INTUNE_COMPLIANCE_ACTION_TYPE_NOTIFICATION - Send notification.
- INTUNE_COMPLIANCE_ACTION_TYPE_BLOCK - Block the device in AAD.
- INTUNE_COMPLIANCE_ACTION_TYPE_RETIRE - Retire the device.
- INTUNE_COMPLIANCE_ACTION_TYPE_WIPE - Wipe the device.
- INTUNE_COMPLIANCE_ACTION_TYPE_REMOVE_RESOURCE_ACCESS_PROFILES - Remove resource access profiles from the device.
- INTUNE_COMPLIANCE_ACTION_TYPE_PUSH_NOTIFICATION - Send push notification to device.
- INTUNE_COMPLIANCE_ACTION_TYPE_REMOTE_LOCK - Remotely lock the device.
