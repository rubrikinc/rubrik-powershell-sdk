### IntunePolicyAssignmentType
Specifies the type of an Intune policy assignment.

- INTUNE_POLICY_ASSIGNMENT_TYPE_UNKNOWN - The type is unknown.
- INTUNE_POLICY_ASSIGNMENT_TYPE_ALL_DEVICES - The type is all devices.
- INTUNE_POLICY_ASSIGNMENT_TYPE_INCLUDE_GROUP - The type is include group.
- INTUNE_POLICY_ASSIGNMENT_TYPE_EXCLUDE_GROUP - The type is exclude group.
- INTUNE_POLICY_ASSIGNMENT_TYPE_ALL_LICENSED_USERS - The type is all licensed users.
