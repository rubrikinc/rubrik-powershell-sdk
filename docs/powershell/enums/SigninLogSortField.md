### SigninLogSortField
Field to sort sign-in logs by.

- SIGNIN_LOG_SORT_FIELD_EVENT_TIMESTAMP - Sort by event timestamp.
- SIGNIN_LOG_SORT_FIELD_ACTOR_PRINCIPAL_NAME - Sort by actor principal name.
- SIGNIN_LOG_SORT_FIELD_RESULT - Sort by result.
- SIGNIN_LOG_SORT_FIELD_PROVIDER - Sort by provider.
- SIGNIN_LOG_SORT_FIELD_RISK_LEVEL - Sort by risk level.
