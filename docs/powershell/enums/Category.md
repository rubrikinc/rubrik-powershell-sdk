### Category
Category of the policy.

- CATEGORY_UNSPECIFIED - DataGov categories
Unspecified category.
- MISPLACED - Data that is stored in an incorrect location.
- REDUNDANT - Data that is duplicated across multiple locations.
- OVEREXPOSED - Data that has excessive access permissions.
- UNPROTECTED - Data that is not adequately protected.
- PRIVILEGED_ACCOUNT_RISK - Identity categories
- IDENTITY_RISK - Issues related to identity risk.
- INFRASTRUCTURE_SECURITY - Issues related to infrastructure security.
- CONFIGURATION_SECURITY - Issues related to configuration security.
- IDENTITY_HYGIENE - Issues related to identity hygiene and management.
- AUTHENTICATION_AND_SECRET_MANAGEMENT - Issues related to authentication and secret management.
- IDENTITY_PROVIDER_SECURITY - Issues related to identity provider security.
- EXCESSIVE_IDENTITY_RIGHTS - Issues related to excessive identity rights.
- MEMBERSHIP_CHANGE - Identity event categories
- GPO_CHANGE - Issues related to GPO changes.
