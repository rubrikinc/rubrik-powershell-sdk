### DevopsConnectionStatus
Connection status for DevOps organization.

- CONNECTION_STATUS_UNSPECIFIED - Default value, should not be used.
- CONNECTION_STATUS_CONNECTED - Organization is connected and accessible.
- CONNECTION_STATUS_DISCONNECTED - Organization is disconnected or not accessible.
- CONNECTION_STATUS_MISSING_PERMISSIONS - Organization has missing permissions.
- CONNECTION_STATUS_CONNECTING - Organization is connecting — added but still being set up.
