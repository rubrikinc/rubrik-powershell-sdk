### IssueEventType
IssueEventType captures what type of event in the issue timeline.

- ADD_WHITELIST_EVENT - Whitelist addition event.
- SNAPSHOT_EVENT - Snapshot event.
- CREATE_EVENT - Issue creation event.
- REMOVE_WHITELIST_EVENT - Whitelist removal event.
- REMOVE_POLICY_OBJ_EVENT - Policy object removal event.
- DELETE_POLICY_EVENT - Policy deletion event.
- NO_ISSUES_SNAPSHOT_EVENT - Snapshot event with no issues.
