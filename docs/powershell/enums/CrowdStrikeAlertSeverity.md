### CrowdStrikeAlertSeverity
CrowdStrike supports 5 severity levels (0-100 scale).
UI dropdown values: INFORMATIONAL, LOW, MEDIUM, HIGH, CRITICAL.

- CROWDSTRIKE_SEVERITY_INFORMATIONAL - Informational severity (0-19).
- CROWDSTRIKE_SEVERITY_LOW - Low severity (20-39).
- CROWDSTRIKE_SEVERITY_MEDIUM - Medium severity (40-59).
- CROWDSTRIKE_SEVERITY_HIGH - High severity (60-79).
- CROWDSTRIKE_SEVERITY_CRITICAL - Critical severity (80-100).
