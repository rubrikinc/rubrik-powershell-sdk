### AzureAdPimGroupAccessType
Group access type for PIM group eligibility.
Mirrors the Microsoft Graph accessId field on
privilegedAccessGroupEligibilitySchedule.

- PIM_GROUP_ACCESS_TYPE_UNSPECIFIED - Default zero value. Indicates the access type was not
set or did not match any known Microsoft Graph value.
- PIM_GROUP_ACCESS_TYPE_MEMBER - Eligibility for group membership.
- PIM_GROUP_ACCESS_TYPE_OWNER - Eligibility for group ownership.
