### QueryFusionComputeVirtualDisksFilterField
Filter fields for querying FusionCompute virtual disks.

- QUERY_FUSION_COMPUTE_VIRTUAL_DISK_FILTER_UNSPECIFIED - Unspecified filter.
- FUSION_COMPUTE_VIRTUAL_DISK_NAME - Disk name (e.g., "i-0000000D-vda").
- FUSION_COMPUTE_VIRTUAL_DISK_VOLUME_UUID - Volume UUID of the disk.
- FUSION_COMPUTE_VIRTUAL_DISK_DATASTORE_URN - Datastore URN of the disk.
