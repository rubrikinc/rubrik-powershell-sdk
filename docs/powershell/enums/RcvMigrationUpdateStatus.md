### RcvMigrationUpdateStatus
Status of RCV migration update operations.

- MIGRATION_UPDATE_STATUS_UNSPECIFIED - Unknown status of the update.
- IN_PROGRESS_MIGRATION_NOT_FOUND - Update failed because no migration is in progress for the location.
- UPDATE_FAILURE - Update failed.
- INVALID_INPUT - Update failed because the provided input is invalid.
- UPDATE_SUCCESSFUL - Update is successful.
