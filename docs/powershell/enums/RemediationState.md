### RemediationState
Describes the state of the remediation.

- REMEDIATION_STATE_UNSPECIFIED - Unspecified remediation state.
- REMEDIATION_STATE_IN_PROGRESS - Remediation is in progress.
- REMEDIATION_STATE_SUCCESS - Remediation was successful.
- REMEDIATION_STATE_FAILED - Remediation failed.
- REMEDIATION_STATE_CLOSED - Remediation was closed.
