### NetworkInterfaceSetting
Supported in v9.4+
Type of actions available for network settings of the restored Domain Controller.

- NETWORK_INTERFACE_SETTING_INTACT - Keep network interface settings intact.
- NETWORK_INTERFACE_SETTING_SOURCE_DC_NETWORK_INTERFACE - Use source domain controller network interface settings.
- NETWORK_INTERFACE_SETTING_TARGET_HOST_NETWORK_INTERFACE - Use target host network interface settings.
