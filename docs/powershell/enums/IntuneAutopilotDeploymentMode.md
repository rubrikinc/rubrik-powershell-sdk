### IntuneAutopilotDeploymentMode
Specifies the deployment mode of an Intune autopilot deployment profile.

- INTUNE_AUTOPILOT_DEPLOYMENT_MODE_UNKNOWN - The deployment mode is unknown.
- INTUNE_AUTOPILOT_DEPLOYMENT_MODE_USER_DRIVEN - User-driven deployment mode.
- INTUNE_AUTOPILOT_DEPLOYMENT_MODE_SELF_DEPLOYING - Self-deploying deployment mode.
- INTUNE_AUTOPILOT_DEPLOYMENT_MODE_PRE_PROVISIONED - Pre-provisioned deployment mode.
