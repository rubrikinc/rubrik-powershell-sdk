### AzureAdAccessReviewFallbackAction
Fallback action taken when a reviewer does not respond.

- ACCESS_REVIEW_FALLBACK_ACTION_UNSPECIFIED - Fallback action is unspecified.
- ACCESS_REVIEW_FALLBACK_ACTION_NO_CHANGE - No change to the user's access.
- ACCESS_REVIEW_FALLBACK_ACTION_REMOVE_ACCESS - Remove the user's access.
- ACCESS_REVIEW_FALLBACK_ACTION_APPROVE_ACCESS - Approve the user's access.
- ACCESS_REVIEW_FALLBACK_ACTION_TAKE_RECOMMENDATIONS - Apply system recommendations.
