### AzurePostgresFlexibleServerSortFields
Fields on which the list of Azure Postgres Flexible Servers can be sorted.

- NAME - Sort Azure Postgres Flexible Servers by name.
- EFFECTIVE_SLA_DOMAIN - Sort Azure Postgres Flexible Servers by SLA Domain.
- AZURE_POSTGRES_FLEXIBLE_SERVER_REGION - Sort Azure Postgres Flexible Servers by region.
- AZURE_POSTGRES_FLEXIBLE_SERVER_RESOURCE_GROUP - Sort Azure Postgres Flexible Servers by resource group.
