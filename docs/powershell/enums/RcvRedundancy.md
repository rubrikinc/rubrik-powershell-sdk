### RcvRedundancy
Redundancy value for the RCV resource.

- REDUNDANCY_UNKNOWN - Rubrik Cloud Vault (RCV) Redundancy Unknown.
- UNKNOWN_AZURE_REDUNDANCY - Rubrik Cloud Vault (RCV) Redundancy Unknown.
- SINGLE_ZONE - Rubrik Cloud Vault (RCV) Single Zone Redundancy.
- AZURE_LRS - Rubrik Cloud Vault (RCV) Single Zone Redundancy.
- MULTI_ZONE - Rubrik Cloud Vault (RCV) Multi Zone Redundancy.
- AZURE_ZRS - Rubrik Cloud Vault (RCV) Multi Zone Redundancy.
- MULTI_REGION - Rubrik Cloud Vault (RCV) Multi Region Redundancy.
- AZURE_GRS - Rubrik Cloud Vault (RCV) Multi Region Redundancy.
