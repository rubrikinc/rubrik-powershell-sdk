### QueryFusionComputeMountsFilterField
Filter field for querying FusionCompute mounts.

- QUERY_FUSION_COMPUTE_MOUNT_FILTER_UNSPECIFIED - Unspecified filter.
- FUSION_COMPUTE_MOUNT_NAME - Name of the FusionCompute mount.
- ORG_ID - Organization ID filter.
- CLUSTER_UUID - Cluster UUID filter.
- ORIGINAL_VM_ID - Source virtual machine FID filter.
