### DevOpsCloudNativeExocompute
DevOpsCloudNativeExocompute is a common exocompute proto for all devops types.

- id: System.String
  - Denotes the ID of the exocompute cloud account associated with the
organization.
- cloudType: CloudServiceProvider
  - Cloud type of the exocompute host.
- hostName: System.String
  - Exocompute host name for the organization.
- region: CloudRegion
  - Exocompute region.
