### LambdaSettings
LambdaSettings holds the anomaly detection settings for an
account.

- anomalyThreshold: System.Single
  - Probability threshold for anomaly detector.
- ransomwareThreshold: System.Single
  - Probability threshold for ransomware detector.
- isAnomalyAlertEnabled: System.Boolean
  - Flag to represent if alert on anomaly workload is enabled.
