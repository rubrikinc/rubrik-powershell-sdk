### AzureAdPimActivePrincipalObject
Principal of a PIM active assignment (user, group, service
principal, or device).

- azureAdUser: AzureAdUser
  - Entra ID user with this PIM active assignment.
- azureAdGroup: AzureAdGroup
  - Entra ID group with this PIM active assignment.
- azureAdServicePrincipal: AzureAdServicePrincipal
  - Entra ID service principal with this PIM active assignment.
- azureAdDevice: AzureAdDevice
  - Entra ID device with this PIM active assignment.
