### Vnet
VNet represents an Azure virtual-network.

- id: System.String
  - The full-path ID for the vnet, it can identify a vnet resource
globally on Azure.
- name: System.String
  - The vnet name.
- regionName: System.String
  - The region the vnet is provisioned in.
- resourceGroup: ResourceGroup
  - The resource group that this VNet is allocated in.
