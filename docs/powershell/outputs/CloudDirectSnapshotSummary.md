### CloudDirectSnapshotSummary
Summary of statistics for a CloudDirect snapshot.

- files: System.Int64
  - Number of files processed.
- dirs: System.Int64
  - Number of directories processed.
- links: System.Int64
  - Number of links processed.
- filesSkipped: System.Int64
  - Number of files skipped.
- dirsSkipped: System.Int64
  - Number of directories skipped.
- linksSkipped: System.Int64
  - Number of links skipped.
- failures: System.Int64
  - Number of failures.
- deleted: System.Int64
  - Number of deleted items.
- bytesProcessed: System.Int64
  - Total bytes processed.
- bytesTransferred: System.Int64
  - Total bytes transferred.
- objectsProcessed: System.Int64
  - Total count of objects processed.
