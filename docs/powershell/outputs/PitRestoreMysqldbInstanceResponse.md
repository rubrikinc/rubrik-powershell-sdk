### PitRestoreMysqldbInstanceResponse
Supported in v9.4+

- id: System.String
  - Required. Supported in v9.4+
ID of the new restore instance created for the point-in-time restore of the MySQL instance.
- asyncRequestStatus: AsyncRequestStatus
  - Required. Supported in v9.4+
Status of the asynchronous job triggered when you initiate the point-in-time recovery operation for the MySQL instance.
