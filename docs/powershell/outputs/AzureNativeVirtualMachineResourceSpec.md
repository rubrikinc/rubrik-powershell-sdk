### AzureNativeVirtualMachineResourceSpec
Resource specification for Azure native virtual machine.

- workloadId: System.String
  - Workload ID.
- workloadName: System.String
  - Name of the workload.
- isArchived: System.Boolean
  - Whether the workload is archived.
- snapshotId: System.String
  - Snapshot ID of the workload.
- sizeType: System.String
  - Size type of virtual machine.
- availabilityZone: System.String
  - Availability zone associated with the virtual machine.
- isAcceleratedNetworkingEnabled: System.Boolean
  - Specifies whether accelerated networking is enabled on the virtual machine.
