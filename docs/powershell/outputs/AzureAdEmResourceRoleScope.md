### AzureAdEmResourceRoleScope
Entitlement Management resource role scope.

- displayName: System.String
  - Display name of the resource.
- resourceType: EmResourceType
  - Type of the resource.
- role: System.String
  - Role granted on the resource (such as "Member", "Owner").
- originId: System.String
  - Identifier of the group or application this scope grants access to.
