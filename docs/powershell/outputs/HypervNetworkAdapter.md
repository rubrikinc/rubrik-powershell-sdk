### HypervNetworkAdapter
Network adapter information for Hyper-V virtual machines.

- macAddress: System.String
  - MAC address of the network adapter.
- adapterName: System.String
  - Name of the network adapter.
- virtualSwitchName: System.String
  - Name of the virtual switch the adapter is connected to.
- ipv4Addresses: list of System.Strings
  - List of IPv4 addresses assigned to the adapter.
- ipv6Addresses: list of System.Strings
  - List of IPv6 addresses assigned to the adapter.
- nicIndex: System.Int32
  - NIC index of the network adapter.
