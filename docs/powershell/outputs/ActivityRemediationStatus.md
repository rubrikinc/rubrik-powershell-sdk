### ActivityRemediationStatus
Remediation status for an activity.

- remediationId: System.String
  - The remediation ID.
- state: RemediationState
  - The state of the remediation.
- createdAt: DateTime
  - The time when the remediation was created.
- remediationType: RemediationType
  - The type of remediation.
- createdById: System.String
  - The user ID of the user who created the remediation.
- createdByEmail: System.String
  - The email of the user who created the remediation.
