### IntuneComplianceScript
Intune device compliance script.

- displayName: System.String
  - Display name of the Intune compliance script.
- operatingSystem: IntuneCompliancePolicyPlatform
  - Operating system platform of the Intune compliance script.
- scriptType: IntuneComplianceScriptType
  - Script type of the Intune compliance script.
- lastModifiedTime: DateTime
  - Date and time when the Intune compliance script was last modified.
- readOnly: System.Boolean
  - Specifies whether the Intune compliance script is read-only.
