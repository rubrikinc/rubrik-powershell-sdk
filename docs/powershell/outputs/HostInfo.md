### HostInfo
Supported in v5.3+

- hostname: System.String
  - Required. Supported in v5.3+
  Name of the host.
- id: System.String
  - Required. Supported in v5.3+
  The managed id of the host.
- oracleQueryUser: System.String
  - Required. Supported in v5.3+
  Oracle discovery user.
- oracleSysDbaUser: System.String
  - Required. Supported in v5.3+
  Oracle sysdba user to use on the host.
