### ProtectionSummaryV2
Response containing the protection summary.

- numWorkloadsCoveredByRecoveryPlan: System.Int32
  - Number of workloads covered by Recovery Plans.
- totalWorkloadsWithSlaProtection: System.Int32
  - Number of workloads protected by an SLA Domain.
- recoveryPlanSummaries: list of AccountRecoveryPlanSummarys
  - Recovery Plan summaries grouped by Recovery Plan type.
