### AwsNativeSubnet
Represents a subnet in AWS.

- id: System.String
  - ID of the subnet.
- name: System.String
  - Name of the subnet.
- availabilityZone: System.String
  - Availability Zone corresponding to the subnet.
- outpostArn: System.String
  - ARN of the AWS Outpost this subnet resides on, if applicable.
