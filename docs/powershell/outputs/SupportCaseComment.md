### SupportCaseComment
A single comment on a support case.

- commentBody: System.String
  - Body text of the comment.
- createdDate: DateTime
  - Timestamp when the comment was created.
- createdByName: System.String
  - Full name of the user who created the comment.
- createdByEmail: System.String
  - Email address of the user who created the comment.
- id: System.String
  - Salesforce CaseComment record ID.
