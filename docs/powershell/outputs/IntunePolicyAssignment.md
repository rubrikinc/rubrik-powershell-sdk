### IntunePolicyAssignment
Intune policy assignment.

- id: System.String
  - ID of the policy assignment.
- policyId: System.String
  - ID of the policy.
- policyName: System.String
  - Name of the policy.
- type: IntunePolicyAssignmentType
  - Type of the assignment.
- groupId: System.String
  - ID of the group assigned to the policy.
- groupName: System.String
  - Name of the group assigned to the policy.
- filterId: System.String
  - ID of the assignment filter applied to the assignment.
- filterName: System.String
  - Name of the assignment filter applied to the assignment.
- filterType: IntuneDeviceAndAppManagementAssignmentFilterType
  - Filter type of the assignment filter applied to the assignment.
