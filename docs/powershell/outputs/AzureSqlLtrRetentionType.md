### AzureSqlLtrRetentionType
AzureSqlLtrRetention specifies the retention duration and unit for
long-term retention (LTR) backups in Azure SQL databases.

- retention: System.Int32
  - Retention specifies the numeric value of the retention period.
- retentionUnit: AzureSqlLtrRetentionUnit
  - Specifies the unit of the retention period (days, weeks,months, or years).
