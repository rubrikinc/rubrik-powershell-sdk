### MysqldbInstanceDetails
MySQL instance user details.

- mysqlUserName: System.String
  - The MySQL instance username.
- systemUserName: System.String
  - The MySQL host system user name.
- bindIpAddress: System.String
  - The MySQL instance bind IP address.
- authenticationType: MysqldbInstanceAuthenticationType
  - The MySQL instance authentication type.
- socketFile: System.String
  - The MySQL instance socket file path.
- sslConfig: MysqldbInstanceSslConfig
  - The MySQL instance SSL configuration.
