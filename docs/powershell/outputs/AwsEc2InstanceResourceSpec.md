### AwsEc2InstanceResourceSpec
Resource specification for the AWS EC2 instance.

- workloadId: System.String
  - Workload ID.
- nativeId: System.String
  - Native ID of the EC2 instance.
- workloadName: System.String
  - Name of the workload.
- instanceType: System.String
  - Instance type of the EC2 instance.
- region: System.String
  - Specifies the region the EC2 instance is in.
- snapshotId: System.String
  - Snapshot ID of the workload.
- isArchived: System.Boolean
  - Whether the workload is archived.
