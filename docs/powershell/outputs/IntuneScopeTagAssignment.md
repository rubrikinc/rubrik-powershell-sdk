### IntuneScopeTagAssignment
Intune scope tag assignment.

- id: System.String
  - ID of the scope tag assignment.
- scopeTagId: System.String
  - ID of the scope tag.
- scopeTagName: System.String
  - Name of the scope tag.
- type: IntunePolicyAssignmentType
  - Type of the assignment.
- groupId: System.String
  - ID of the group assigned to the scope tag.
- groupName: System.String
  - Name of the group assigned to the scope tag.
- filterId: System.String
  - ID of the assignment filter applied to the assignment.
- filterName: System.String
  - Name of the assignment filter applied to the assignment.
