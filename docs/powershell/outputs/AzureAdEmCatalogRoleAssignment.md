### AzureAdEmCatalogRoleAssignment
Role assignment granting a principal access to manage an Entitlement Management catalog or its access packages.

- displayName: System.String
  - Display name of the assigned principal.
- principalType: AzureAdRoleAssignmentPrincipalType
  - Type of the principal.
- userPrincipalName: System.String
  - User principal name of the assigned user (if the principal is a user).
- role: EmCatalogRole
  - Catalog role assigned to the principal.
