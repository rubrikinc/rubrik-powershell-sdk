### ActiveDirectoryGpoSettingsData
Supported in v9.4+

- data: System.String
  - Required. Supported in v9.4+
GPO settings data in html or xml format.
- domainSid: System.String
  - Required. Supported in v9.4+
SID of the Active Directory domain.
- gpoId: System.String
  - Required. Supported in v9.4+
ID of the GPO for which settings needs to be retrieved.
- snapshotId: System.String
  - Required. Supported in v9.4+
Snapshot ID of the GPO from which settings are retrieved.
