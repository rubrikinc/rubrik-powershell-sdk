### IntuneAutopilotDeploymentProfile
Intune autopilot deployment profile.

- displayName: System.String
  - Display name of the deployment profile.
- deploymentMode: IntuneAutopilotDeploymentMode
  - Deployment mode of the profile.
- joinType: IntuneAutopilotDeploymentProfileJoinType
  - Join type of the profile.
- lastModifiedDateTime: DateTime
  - Date and time when the profile was last modified.
- roleScopeTagNames: list of System.Strings
  - Role scope tag names associated with the profile.
- roleScopeTagIds: list of System.Strings
  - Role scope tag IDs associated with the profile.
