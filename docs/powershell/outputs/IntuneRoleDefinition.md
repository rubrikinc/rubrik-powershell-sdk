### IntuneRoleDefinition
Intune role definition.

- displayName: System.String
  - Display name of the role definition.
- roleType: System.String
  - Type of the role (built-in or custom).
- description: System.String
  - Description of the role definition.
- roleScopeTagNames: list of System.Strings
  - Role scope tag names associated with the role definition.
- roleScopeTagIds: list of System.Strings
  - Role scope tag IDs associated with the role definition.
