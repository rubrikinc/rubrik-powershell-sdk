### AssignedRscTag
Details of an RSC tag.

- description: System.String
  - Description of the RSC tag.
- id: System.String
  - ID of the RSC tag.
- isArchived: System.Boolean
  - Specifies whether the tag is archived.
- key: System.String
  - Key of the RSC tag (empty for single-value tags).
- lastModified: System.String
  - Time the RSC tag was last modified.
- name: System.String
  - Name of the RSC tag.
- slaDomainId: System.String
  - SLA Domain ID applied to the RSC tag.
- value: System.String
  - Value of the RSC tag. (For single-value tags, only this field is set).
