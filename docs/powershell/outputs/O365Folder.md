### O365Folder
An O365 mailbox folder.

- id: System.String
  - The ID of the folder.
- name: System.String
  - The display name of the folder.
- parentFolderId: System.String
  - The parent folder ID of the folder (ROOT indicates root folder).
- snapshotNum: System.Int32
  - The snapshot number of this version of the folder.
- hierarchyType: ExchangeItemHierarchyType
  - Type of hierarchy for the specified folder.
- snapshotId: System.String
  - The snapshot ID of this version of the folder.
