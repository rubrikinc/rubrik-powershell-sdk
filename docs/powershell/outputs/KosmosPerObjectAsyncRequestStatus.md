### KosmosPerObjectAsyncRequestStatus
Supported in v9.6
Per-object async request status for a Kosmos automated restore. One entry per restore object for snappables that fan out per object.

- id: System.String
  - Required. Supported in v9.6
ID of the restore instance for this entry.
- asyncRequestStatus: AsyncRequestStatus
  - Required. Supported in v9.6
Status of the asynchronous job for this entry.
