### DevOpsBackupLocation
Backup location for DevOps organization.

- id: System.String
  - ID of the backup location.
- name: System.String
  - Name of the backup location.
- archivalGroupId: System.String
  - ID of the archival group.
- cloudType: CloudServiceProvider
  - Cloud type of the backup location.
- storageType: DevOpsStorageType
  - Storage type of the backup location.
- cloudSpecificRegion: CloudSpecificRegionOneof
  - Region can be any one of CloudSpecificRegion.
