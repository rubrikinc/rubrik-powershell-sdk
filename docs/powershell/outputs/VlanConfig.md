### VlanConfig
Supported in v5.0+

- interfaces: list of NodeIps
  - Required. Supported in v5.0+
  Network interfaces for each node.
- netmask: System.String
  - Required. Supported in v5.0+
  Netmask for addresses on this VLAN.
- vlan: System.Int32
  - Required. Supported in v5.0+
- gateway: System.String
  - Supported in v9.2+
  Gateway for the VLAN.
- alias: System.String
  - Supported in v9.2+
  Alias for the VLAN.
