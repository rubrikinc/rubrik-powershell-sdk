### SaasAppsStorageLocation
Storage location details for the SaaS app organization's backed up data.

- storageRegion: System.String
  - The region where the storage is deployed.
- appRegions: list of System.Strings
  - List of application regions that map to the storage region.
- isDefault: System.Boolean
  - Indicates whether this is the default storage location for the organization backed up data.
