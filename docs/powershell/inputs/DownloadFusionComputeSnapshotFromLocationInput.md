### DownloadFusionComputeSnapshotFromLocationInput
Input for downloading a FusionCompute snapshot from a specific location.

- id: System.String
  - Required. ID of the snapshot to be downloaded.
- downloadConfig: FusionComputeSnapshotDownloadRequestInput
  - Configuration for the download job.
- locationId: System.String
  - Required. ID of the archival or replication location.
