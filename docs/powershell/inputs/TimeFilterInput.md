### TimeFilterInput
Time range filter for report queries.

- endTime: System.String
  - End time of the range as an RFC3339 timestamp.
- startTime: System.String
  - Start time of the range as an RFC3339 timestamp.
- timeDuration: TimeDuration
  - Represents the size of our time intervals.
- timezone: System.String
  - Timezone represents the conversion needed before returning timestamps.
