### MysqldbSslConfigInput
Supported in v9.4+
SSL configuration of the MySQL instance.

- certFilePath: System.String
  - Supported in v9.4+
Path to the SSL certificate file.
- caCertFilePath: System.String
  - Supported in v9.4+
Path to the Certificate Authority (CA)-signed certificate file.
- keyFilePath: System.String
  - Supported in v9.4+
Path to the SSL key file.
