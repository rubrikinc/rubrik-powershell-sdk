### CreateActiveDirectoryDownloadFilesJobInput
Input for creating a job to download files from an Active Directory Domain Controller snapshot.

- id: System.String
  - Required. ID assigned to an Active Directory Domain Controller snapshot.
- config: ActiveDirectoryDownloadFilesJobConfigInput
  - Required. Configuration information for a job to download files and folders from an Active Directory Domain Controller snapshot.
- userNote: System.String
  - User note to associate with audits.
