### CloudDirectGlobalSearchReq
CloudDirectGlobalSearchReq represents inputs for CloudDirectGlobalSearch.

- clusterUuid: System.String
  - UUID of the NAS Cloud Direct cluster to search.
- filter: System.String
  - Search filter pattern for prefix search.
- marker: System.String
  - Pagination marker from previous response.
- prefix: System.String
  - Path prefix filter to narrow down search scope.
