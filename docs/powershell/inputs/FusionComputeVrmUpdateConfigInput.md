### FusionComputeVrmUpdateConfigInput
Configuration for updating a FusionCompute Virtual Resource Management (VRM) instance.

- caCerts: System.String
  - Optional CA certificate content for SSL verification.
- endpointUri: System.String
  - Required. The address of the updated FusionCompute VRM instance.
- password: System.String
  - Required. The password for authenticating with the updated FusionCompute VRM instance.
- username: System.String
  - Required. The username of the updated FusionCompute VRM instance.
