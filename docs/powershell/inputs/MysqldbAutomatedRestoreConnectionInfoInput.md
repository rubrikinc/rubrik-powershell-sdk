### MysqldbAutomatedRestoreConnectionInfoInput
Supported in v9.5+
MySQL credentials to be used during recovery.

- password: System.String
  - Required. Supported in v9.5+
Source/Target instance restore password for accessing the MySQL instance for recovery.
- username: System.String
  - Required. Supported in v9.5+
Source/Target instance restore username for accessing the MySQL instance for recovery.
